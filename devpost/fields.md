## Short Description

```text
Edgion Atlas prepares the messy middle of a physical-risk review. The demo uses a synthetic data-centre case to show source checks, spatial and temporal context, missing resilience evidence, a human review action, and a decision record. UiPath controls the case path around that preparation.
```

## Safe Core Line

```text
UiPath owns the governed case workflow; Edgion agents provide the specialist work-up; the human reviewer owns the business decision.
```

## Boundary Line

